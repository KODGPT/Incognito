# 🕶 Open in Incognito

Лёгкое **Chrome расширение**, которое позволяет **открыть текущую страницу в режиме инкогнито одним кликом**.

Полезно если вы хотите быстро:

* открыть сайт **в приватном режиме**
* проверить страницу **без cookies и авторизации**
* открыть ссылку **в incognito**
* протестировать сайт **как новый пользователь**

---

## ⚡ Возможности

* 🕶 **Открывает текущую вкладку в режиме инкогнито**
* ⚡ **Работает одним кликом по иконке**
* 🌐 Открывает **ту же страницу (тот же URL)**
* 🪶 **Минимальный и быстрый код**
* 🔒 **Не собирает и не отправляет данные**

---

## 🔎 Как это работает

Когда вы нажимаете на иконку расширения:

1. Расширение получает **URL текущей вкладки**
2. Создаёт **новое окно браузера**
3. Открывает страницу **в режиме Incognito**

Пример:

Обычная вкладка
https://example.com

↓

Новое окно
Incognito mode
https://example.com

---

## 🧰 Когда это полезно

Расширение удобно если нужно:

* проверить сайт **без сохранённых cookies**
* протестировать **логин / регистрацию**
* открыть страницу **как новый пользователь**
* проверить **кэш и авторизацию**
* быстро перейти в **приватный режим**

---

## 🔒 Безопасность

Расширение:

* ❌ не читает содержимое страниц
* ❌ не отслеживает пользователей
* ❌ не отправляет данные на сервер

Оно только **открывает текущий URL в окне инкогнито**.

---

## 📦 Установка

1. Скачать репозиторий

git clone https://github.com/yourusername/open-in-incognito

2. Открыть страницу расширений Chrome

chrome://extensions

3. Включить **Developer mode**

4. Нажать **Load unpacked** и выбрать папку проекта.
